AI Sales Analytics Agent
A powerful AI-driven Sales Analytics platform designed to help organizations analyze sales data, identify trends, and make smarter business decisions. The system uses Generative AI and Data Analytics to provide insights such as sales performance, customer behavior patterns, forecasting, and automated reports. It integrates AI models to process sales datasets and deliver meaningful business intelligence in real time.

📦 Requirements
To run this project locally, you need the following installed:
Node.js (v18.x or higher)
npm (v9.x or higher)
MongoDB Database (Local or Atlas Cluster)
Python (if using ML models)
API Keys (if using external AI services)
Environment Variables (.env)
Add the following variables in your backend .env file:
Copy code

MONGO_URI=your_mongo_database_uri
AI_MODEL_API_KEY=your_ai_api_key
DATA_SOURCE_URL=your_sales_data_source
PORT=5000
🛠️ Step-by-Step Instructions to Run Locally
1. Clone the Repository
Copy code

git clone https://github.com/your-username/ai-sales-analytics-agent.git
cd ai-sales-analytics-agent
2. Install Dependencies
Since this is a Node.js project, dependencies are installed using:
Copy code

npm install
3. Setup the Backend
The backend handles:
Sales data processing
AI analytics generation
Data storage
API responses
Copy code

cd backend
npm install
Create a .env file:
Copy code

touch .env
Add environment variables.
Start the backend server:
Copy code

npm start
Backend will run on:
Copy code

http://localhost:5000
4. Setup the Frontend
The frontend provides an interactive dashboard to visualize analytics.
Copy code

cd frontend
npm install
Create .env file:
Copy code

VITE_API_URL=http://localhost:5000
Run the frontend:
Copy code

npm run dev
Open in browser:
Copy code

http://localhost:3000
